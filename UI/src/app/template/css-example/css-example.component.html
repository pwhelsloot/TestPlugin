CSS Example

Once component styles have been loaded they will not be removed until an application refresh.
This is because angular loads component styles when they are created but doesn't remove them when destroyed
All components styles should be Encapsulated. This ensures styles cannot affect other components.
Using ViewEncapsulation.None allows your component styles to affect every other loaded component in the entire component tree.
This leads to seriously difficult to re-create bugs how do you tell which component in the tree is bleeding into others?
Need to style a component differently on a particular screen? Expose a new input on the component!
